# Distributed-vending-machine

## Intro
RUP(Rational Unified Process)를 기반으로 한 개발방법론인 OOPT를 적용한 분산 자판기 시스템 구축

## Stack
- Java 21
- MySQL
- Test : JUnit5, Mockito, H2
- CI/CD : Jenkins
